TITLE
  Nigeria LGA-Level Population and Socioeconomic Indicators
  (2010–2024)

OVERVIEW
  This dataset provides a multi-year panel of demographic,
  economic, and social indicators at the Local Government Area
  (LGA) level for five Nigerian states: Lagos, Rivers, Enugu,
  Kano, and Kaduna. It spans the period 2010–2024 and covers
  20 LGAs with 25 indicator variables per record.

  Variables include population size and structure, urban/rural
  split, birth and death rates, migration, literacy, education,
  employment, healthcare access, poverty, and housing deficit.
  It is suited for longitudinal socioeconomic analysis,
  clustering, and transport demand modelling.

FILE FORMAT
  Format      : CSV (comma-delimited)
  Encoding    : UTF-8
  Rows        : 200 (excluding header)
  Columns     : 25

SCHEMA
  Field                      Type     Description
  --------------------------  -------  ----------------------------
  Year                       int      Survey/reference year (2010–2024)
  State                      string   Nigerian state name
  LGA                        string   Local Government Area name
  Population                 int      Total LGA population estimate
  Area_sqkm                  float    LGA area in square kilometres
  Population Density         float    Persons per sq km
  Urban %                    float    Share of population in urban areas (%)
  Rural %                    float    Share of population in rural areas (%)
  Male Population            int      Estimated male population
  Female Population          int      Estimated female population
  Youth %                    float    Share of population aged 15–35 (%)
  Elderly %                  float    Share of population aged 60+ (%)
  Growth Rate                float    Annual population growth rate (%)
  Birth Rate                 float    Crude birth rate per 1,000 population
  Death Rate                 float    Crude death rate per 1,000 population
  Migration Rate             float    Net migration rate (negative = net emigration)
  Median Age                 float    Median age of population (years)
  Urbanization Level         float    Urbanisation index score (%)
  Literacy %                 float    Adult literacy rate (%)
  Education Index            float    Composite education index (0–1 scale)
  Employment Rate            float    Share of working-age population employed (%)
  Unemployment %             float    Unemployment rate (%)
  Healthcare Access          float    Share of population with healthcare access (%)
  Poverty Rate %             float    Share of population below poverty line (%)
  Housing Deficit            int      Estimated housing unit shortfall
