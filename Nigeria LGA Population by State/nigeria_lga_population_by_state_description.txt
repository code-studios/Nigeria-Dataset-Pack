TITLE
  Nigeria LGA Population by State

OVERVIEW
  This dataset provides Local Government Area (LGA) level
  population figures for Nigerian states. The original source
  file was structured in a wide side-by-side format (one pair
  of columns per state), which has been reshaped into a clean
  long-format table (State, LGA, Population) for analysis.

  The Excel workbook contains two sheets:
    1. "All LGAs"      — full long-format record of all LGAs
    2. "State Summary" — aggregated stats per state (LGA count,
                         total population, mean, min, max)

  The original wide-format file is also preserved as:
    nigeria_lga_population_by_state_original_wide_format.xlsx

FILE FORMAT
  Primary   : nigeria_lga_population_by_state.xlsx  (reshaped, long format)
  Secondary : nigeria_lga_population_by_state_original_wide_format.xlsx  (original wide format)
  Encoding  : UTF-8

SCHEMA  (All LGAs sheet)
  Field        Type    Description
  -----------  ------  ----------------------------------------
  State        string  Nigerian state or FCT name
  LGA          string  Local Government Area name
  Population   int     LGA population estimate

INTENDED USE

LIMITATIONS
