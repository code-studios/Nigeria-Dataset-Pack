PURPOSE
  This file documents the origin, confidence level, and
  any known limitations for each dataset in the collection.
  Datasets are rated on a 3-tier source confidence scale:

    ✓ CONFIRMED   — Source is known, citable, and verifiable.
    ~ PROBABLE    — Source is likely identified but not
                    fully confirmed (e.g. platform known,
                    exact URL or edition unclear).
    ? UNCONFIRMED — Source is uncertain or not traceable.
                    Data values may be estimated or synthetic.

  Users citing these datasets in academic publications should
  verify unconfirmed sources independently before citing.

08  Nigeria Inter-City Travel and Traffic Data
    File : nigeria_intercity_travel_traffic_data.csv
    ··················································
    Confidence : ? UNCONFIRMED — likely synthetic
    Origin : Source unknown. No author, URL, or agency
             attribution is available.
    Evidence of synthetic generation:
      1. Road length values have high floating-point
         precision (e.g. 772.2622439999999 km) inconsistent
         with measured road data.
      2. The 896 records represent a near-perfectly
         balanced 9×9 city matrix with ~11 observations
         per OD pair — unlikely in empirical data.
      3. Weather categories (Clear/Cloudy/Rainy/Foggy)
         appear randomly distributed rather than tied to
         any meteorological record.
    Use guidance: Suitable for methodological illustration,
    model testing, and sensitivity analysis. Do NOT cite
    as empirical traffic data in publications without
    independent verification against FRSC, FMWORKS, or
    Google Maps Distance Matrix data.

09  Railway Station Optimization Results
    Sub-folders: 01–11 (see README_DATASET_INDEX.txt)
    ··················································
    Confidence : ✓ CONFIRMED (derived/computed)
    All optimization outputs were produced from the above
    raw datasets using Python scripts (available on request
    from the maintainer). Key derivation notes:

    Distance matrix
      Primarily Haversine great-circle distances multiplied
      by a 1.3 road correction factor. Where actual road
      distances were available from Dataset 08, those values
      were used instead. Check distance_type_matrix.csv
      to identify which values are "actual" vs "estimated".

    Demand matrices
      Synthetic — computed via gravity model from population
      and distance data. Not observed passenger counts.

    Corrected metrics
      Two systematic errors were found in the original
      optimization run and corrected in stage2_issue_audit.py:
        1. Network length was computed as sum of all
           pairwise distances instead of MST edges only.
        2. Scenario 1 avg distance to station was
           incorrectly non-zero.
      Use corrected_metrics.json (Group 07) for all
      publication figures.

--------------------------------------------------------------------
  GENERAL GUIDANCE FOR PUBLICATION USE
--------------------------------------------------------------------

  Before submitting any work citing these datasets:

  1. VERIFIED SOURCES (Datasets 02, 04):
     Cite the NBS/NDHS/FRSC sources directly as listed
     in the dataset description files. No further
     verification required.

  2. PROBABLE SOURCES (Datasets 01, 07):
     Attempt to locate and cite the original source.
     Suggested approach:
       - Search NBS Nigeria portal for matching values
       - Search World Bank Open Data for Nigeria indicators
       - If unlocatable, note as "Source unverified;
         values consistent with NBS estimates"

  3. UNCONFIRMED SOURCES (Datasets 03, 05, 06, 08):
     Do not cite as authoritative data sources. Instead:
       - Use only as inputs for methodological illustration
       - Clearly state in the paper: "indicative estimates"
         or "model-generated data"
       - For Datasets 03 & 05: cross-check coordinates
         against GeoNames.org or NGA boundary files
       - For Dataset 08: replace with FRSC or Google Maps
         Distance Matrix data for empirical validation

  4. OPTIMIZATION RESULTS (Dataset 09):
     Fully reproducible from the provided Python scripts.
     Cite as computed outputs; reference the methodology
     paper for the algorithmic details.
